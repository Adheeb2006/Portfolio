# My Portfolio

A modern, responsive portfolio website built with Bootstrap 5.

## Features

- Responsive design that works on all devices
- Smooth scrolling navigation
- Animated sections with fade-in effects
- Contact form
- Skills showcase
- Project gallery
- Social media links

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Bootstrap Icons

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Customize the content to match your information

## Customization

- Replace placeholder images with your own
- Update the text content in `index.html`
- Modify colors and styles in `style.css`
- Add your own projects and skills

## Deployment

This is a static website, so you can deploy it to any static hosting service like GitHub Pages, Netlify, or Vercel.
